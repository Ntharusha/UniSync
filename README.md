# UniSync — University Appointment System

**UniSync** is a cross-platform appointment scheduling system purpose-built for universities. It replaces the informal, email-based process of booking student–lecturer meetings with a structured, timetable-aware, role-based workflow.

## 🌟 Key Features
- **Smart Availability Engine**: Subtracts teaching blocks and blackout periods from lecturer office hours to show true availability.
- **Priority Override System**: Automatically manages competing booking requests based on urgency (Normal, Urgent, Emergency).
- **Socket.io Integration**: Real-time slot updates and instant notifications.
- **Role-Based Access**: Specialized dashboards for Students, Lecturers, and Administrators.
- **Premium Design**: Modern, responsive UI built with Tailwind CSS and Framer Motion.

## 📂 Project Structure
- `/backend`: Node.js/Express API with MongoDB/Mongoose.
- `/frontend`: React/Vite application with Tailwind CSS.
